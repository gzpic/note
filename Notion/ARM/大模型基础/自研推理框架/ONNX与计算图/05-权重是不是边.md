# 权重是不是边

## 1. 结论

是的。

在计算图抽象里，权重本质上也是边，因为它本质上也是 Tensor。

更准确地说：

> 权重是常量边（constant tensor edge）。

## 2. 普通边和权重边的区别

普通边通常表示运行时数据，例如：

- 输入
- 中间结果
- 输出

而权重边表示常量数据，例如：

- Linear 权重
- bias
- embedding table

它们本质上都是 Tensor，只是来源不同：

- 普通边：runtime tensor
- 权重边：constant tensor

## 3. 例子

对于：

```text
Y = X @ W + b
```

图结构可以写成：

```text
      W
      ↓
X → MatMul → A → Add → Y
                ↑
                b
```

这里：

- `X` 是输入边
- `A` 是中间边
- `W` 和 `b` 也是边，只不过是常量边

## 4. 在 ONNX 里的体现

在 ONNX 里，权重通常会出现在 `initializer` 中，但节点引用它们的方式和普通输入张量是一致的。

所以结构上仍然可以理解成：

```text
W → MatMul
```

这就是为什么说权重本质上也是边。

## 5. 面试速记版

- 权重本质上也是 Tensor
- Tensor 在图结构里就是边
- 所以权重可以看作常量边
