3.MLP，h1→h2

① LayerNorm

③ 残差

h2=h1+FFN(x)

# ⭐ 3. logits 是什么？

logits 是：

```plain text
每个词（vocab）对应的未归一化分数
shape = [vocab_size]


```

例如 vocab = 151k，那 logits 就有 151k 个值。

decode阶段生成的logits size =  [batch_size, vocab_size],

prefill 输出 logits 的 size = **[T, **vocab_size**],但只用最后一个v，用于生成下一个token**
