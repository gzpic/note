定目标：

### 时延类

- **TTFT（Time To First Token）**
    - 首 token 延迟
    - 用户体感最强
- **单 token decode latency**
    - 每生成一个 token 要多久
- **端到端响应时间**
    - 从输入到输出完成总耗时
- **Prefill latency**
    - 输入长 prompt 时的处理时间

### 吞吐类

- **Tokens/s**
    - 最常用
    - 一般分：
        - prefill tokens/s
        - decode tokens/s
- **QPS / requests per second**
    - 多请求场景才更重要
