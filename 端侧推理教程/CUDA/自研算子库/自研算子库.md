[[p和d的执行算子]]

1. 基础的算子、

GEMM, ELEMENTWISE,Reduce,

| 基础算子 | 重要性 | 性能敏感点 / 备注 | 推荐优化手段 |
| --- | --- | --- | --- |
| GEMM（矩阵乘法） | ⭐⭐⭐⭐⭐ | 核心计算密集，prefill & decode 都用 | Tensor Core；FP16 / INT8；融合 bias |
| Load / Store | ⭐⭐⭐⭐⭐ | KV Cache / Embedding 的内存访问 | 连续内存；避免 memcpy；固定 stride |
| Reduce（sum / max） | ⭐⭐⭐⭐ | Softmax / RMSNorm / Decode Attention | 与 elementwise 融合；避免多 kernel launch |
| Elementwise (Add / Mul / FMA) | ⭐⭐⭐⭐ | Residual / FFN / SiLU / GLU | 融合进 Linear / FFN / RMSNorm |
| Exp / RSqrt | ⭐⭐⭐⭐ | Softmax / RMSNorm / SiLU | 内联到 fused kernel；FP16 math |
| Gather | ⭐⭐⭐ | Embedding / KV Read | 连续内存 + coalesced access |
| Compare / Select | ⭐⭐ | Causal Mask | 尽量隐式 mask，减少计算 |
| Random / PrefixSum | ⭐ | Sampling / Top-p | CPU 可做，GPU kernel 可选 |

2.常用的算子如何用基础算子来生成的呢都

组合+融合

| 高阶算子 | 使用阶段 | 重要性 | 基础算子（你真正要实现的） | Orin Nano 推荐优化手段 |
| --- | --- | --- | --- | --- |
| Token Embedding | P / D | ⭐⭐⭐⭐⭐ | `Gather` / `Load` | GPU lookup；连续表 |
| RMSNorm | P / D | ⭐⭐⭐⭐⭐ | `ReduceSum` + `RSqrt` + `Mul` | **与 Linear 融合** |
| QKV Linear | P / D | ⭐⭐⭐⭐⭐ | `GEMM` + `AddBias` | **QKV 合并 GEMM** |
| RoPE | P / D | ⭐⭐⭐⭐⭐ | `SinCos` + `Mul` + `Add` | **融合进 QKV** |
| Attention（Prefill） | P | ⭐⭐⭐⭐ | `GEMM(QKᵀ)` + `Scale` + `Softmax` + `GEMM(@V)` | 普通实现即可 |
| Attention（Decode） | D | ⭐⭐⭐⭐⭐ | `Dot` + `ReduceMax` + `Exp` + `ReduceSum` + `MulAdd` | **专用 decode kernel** |
| Softmax | P / D | ⭐⭐⭐⭐⭐ | `ReduceMax` + `Exp` + `ReduceSum` + `Div` | **融合进 attention** |
| KV Cache Write | P / D | ⭐⭐⭐⭐⭐ | `Store` / `Memcpy` | 连续 append |
| KV Cache Read | D | ⭐⭐⭐⭐⭐ | `Load` / `Gather` | 紧凑 layout |
| Attn Out Linear | P / D | ⭐⭐⭐⭐ | `GEMM` + `AddBias` | FP16 GEMM |
| FFN Linear ×2 | P / D | ⭐⭐⭐⭐⭐ | `GEMM` + `AddBias` | Tensor Core |
| SiLU | P / D | ⭐⭐⭐⭐ | `Mul` + `Sigmoid` | **与 Linear 融合** |
| GLU Mul | P / D | ⭐⭐⭐⭐ | `Mul` | 融合进 FFN |
| FFN Down Linear | P / D | ⭐⭐⭐⭐⭐ | `GEMM` | FP16 / INT8 |
| Residual Add | P / D | ⭐⭐⭐ | `Add` | 融合 |
| Causal Mask | P | ⭐⭐⭐⭐ | `Compare` + `Select` | **隐式 mask** |
| LM Head Linear | D | ⭐⭐⭐⭐⭐ | `GEMM` + `AddBias` | **Top-K logits** |
| Argmax / TopK | D | ⭐⭐⭐⭐ | `ReduceMax` / `PartialSort` | GPU kernel |
| Sampling | D | ⭐⭐⭐ | `Random` + `PrefixSum` | 简单实现即可 |
| Dequant / Scale | 全程 | ⭐⭐⭐ | `Mul` / `FMA` | 融合进 GEMM |
| CUDA Graph | D | ⭐⭐⭐⭐ | （不是算子） | 固定 shape |
