# 高频面试考点

这套目录是从现有 `[[Notion/面试整理/README]]`、专题笔记和项目整理里再抽一层，目标只有一个：

- 只保留高频考点
- 只保留面试里最稳的答法
- 尽量让每页都能在 5 到 10 分钟内快速复习

## 使用方式

建议按下面顺序看：

1. 先过本页，知道有哪些领域
2. 再看对应领域页里的“高频题”和“回答骨架”
3. 最后回到原始专题笔记补细节

## 这套目录和原笔记的分工

- 原目录 `[[Notion/面试整理/README]]` 更像系统整理版
- 当前目录更像高频速记版
- 原专题页保留完整上下文
- 当前页优先保留“怎么答得稳”

## 整理原则

- 每个领域只放高频问题，不追求全
- 每个问题优先写“一句话结论 + 三四个回答抓手”
- 能直接口述的答案优先
- 细节、例子、展开原理尽量回链到原笔记

## 建议后续补法

- 每轮面试后，把新被问到的问题补到对应页
- 高频但自己答不稳的问题，单独加“追问补充”
- 只补真实会被问到的，不把这里变成第二套大而全目录
